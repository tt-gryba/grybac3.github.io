Date Selection panel

On the left of the screen, the Date Selection panel shows the daily activity logs that have been selected using the date picker and are available for data visualization.

  1. Date picker: Date-based selector to add daily activity data into the Date Selection panel.
  2. Selected Dates: Dated logs that are included in the current chart.
  3. Recently Unloaded: Dated logs that have been added to TT Score but are not included in the current view.
