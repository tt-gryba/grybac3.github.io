TT Score divides audit log data into sections called clusters. TT Score's machine learning algorithm compares these clusters to models of known problematic trading activity called cortices and generates a numeric score to indicate that cluster's similarity to a particular cortex.

  • A cortex is a computer-generated model of a particular type of problematic trading pattern.

What is a cortex?

A neural cortex is a -descriptor- model of a specific problematic trading pattern to which a trader's activity is compared and evaluated for similarity. TT Score uses a number of models, each representing a different trading pattern that may draw regulatory attention. Each cortex uses its ownclustering algorithm to divide a trader's audit log into chunks of related trading activity. These clusters are compared to the cortex and each given an individual score.

Machine learning: training a cortex

Each cortex is created using training data which provides the seed patterns that each model utilizes to identify similar disruptive or manipulative trading behavior. This training data is taken from real-world regulatory data from successfully prosecuted cases. When regulatory data is unavailable for a particular type of manipulative or disruptive trading activity, we train the models utilizing customized training data developed by several domain experts with backgrounds in both the business and regulatory sides of the financial industry.
